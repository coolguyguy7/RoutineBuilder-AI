# RoutineBuilder-AI

**Live Demo:** https://attached-assets--ethanguy77.replit.app/

An AI-powered app that helps users build healthier, more productive summer routines.

## What It Does

RoutineBuilder-AI creates personalized summer routines based on a user's goals, interests, and daily schedule. Whether you want to stay active, learn a new skill, or improve your productivity, the app generates practical, customized plans that are easy to follow and adjust over time.

## Stack

* Frontend: HTML, CSS, JavaScript
* AI: OpenAI API
* Deployment: Replit
